팀홈페이지 설계


[기존 설계안]

위 슬라이드와 같이 슬라이드가 중심이 되는 홈페이지 메인에서 원페이지 방식의 메인으로 계획을 변경했습니다.



팀홈페이지 사용 기술 및 기능

메인 홈페이지

· 자바스크립트를 이용한 버튼을 클릭하면 지정한 섹션으로 스크롤이 이동하는 기능 구현

· 세로 값을 기준으로 한 텍스트 애니메이션 구현

· 측면의 사이드바에 position: fixed; 값을 주어 항상 화면 위에 위치하게 함


게임 홈페이지

· Grid 기능을 이용한 디자인

· 자바스크립트를 이용하여 이미지 슬라이드 기능 구현

· 부트스트랩을 이용하여 드롭다운 네비게이션 구현
